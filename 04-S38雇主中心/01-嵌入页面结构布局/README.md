# 1. 财税页面layout不显示头部底部

`nuxtjs` 中的 `layout` 表示的是布局，`layout` 中的文件能够引用一些公共的组件，默认是 `default.vue` ，当然也可以自定义

具体引用layout 中的哪一个布局，请在 pages 路径下的目标 index.vue中 添加设置

`layout: "自定义的layout中的文件名"`

* 将财税的测试环境的链接添加到集团链接的后方
